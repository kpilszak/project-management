## General info
This project is a Spring Boot application.  
The main purpose is to create a project management app.  

## Technologies
Project is created with:
* Java 8 
* Spring Boot 2.4.1
* Spring Security
* Spring MVC
* Spring Data
* Hibernate
* Thymeleaf  
  
IDE: Intellij IDEA 2020
